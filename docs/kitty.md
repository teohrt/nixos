# Kitty Keybindings

Keybindings mirror Hyprland, using `Ctrl` instead of `Super`.

## Window (Split) Management

| Key | Action |
|-----|--------|
| `Ctrl+Enter` | New split (inherits cwd) |
| `Ctrl+Q` | Close window |
| `Ctrl+F` | Toggle zoom (fullscreen pane) |
| `Ctrl+J` | Rotate splits |

## Window Focus

| Key | Action |
|-----|--------|
| `Ctrl+Left` | Focus left |
| `Ctrl+Right` | Focus right |
| `Ctrl+Up` | Focus up |
| `Ctrl+Down` | Focus down |

## Window Movement

| Key | Action |
|-----|--------|
| `Ctrl+Shift+Left` | Move window left |
| `Ctrl+Shift+Right` | Move window right |
| `Ctrl+Shift+Up` | Move window up |
| `Ctrl+Shift+Down` | Move window down |

## Tabs

| Key | Action |
|-----|--------|
| `Ctrl+Shift+Enter` | New tab (inherits cwd) |
| `Ctrl+1-9` | Switch to tab 1-9 |
| `Ctrl+Shift+T` | Detach window to new tab |
| `Ctrl+T` | Rename tab |

## Clipboard

| Key | Action |
|-----|--------|
| `Super+C` | Copy |
| `Super+V` | Paste |
